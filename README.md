# 양노환장 내부 RPG 플러그인
반갑다 게이야 양노환장 마인크래프트 서버에서 쓸 플러그인이노 ㅋㅋ

## 기능
- 재화
- 직업
    - 전사
    - 마법사
    - 궁수
- 상점
    - 아이템 가격 변동
- 능력치
    - 공격력
    - 주문력
    - 방어력
    - 체력
    - 이동속도
    - 치명타 확률, 데미지
- 스킬
    - {스킬북}
- 경험치
    - 낚시 경험치
    - 농사 경험치
    - 전투 경험치
    - 채광 경험치
- 아이템
    - 검
    - 지팡이
    - 활
    - 농작물
    - 생선
    - 강화석
- 강화
